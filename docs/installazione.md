# Installazione

## Requisiti
- Browser moderno (Chrome, Firefox, Edge, Safari)
- Connessione internet per caricare Bootstrap

## Come aprire il progetto
1. Clona la repository:
   ```bash
   git clone https://github.com/Halocavo92/DashBoard-Meteo.git
   cd DashBoard-Meteo
   ```
2. Apri `index.html` nel browser.

## Se usi Visual Studio Code
Se hai installato l'estensione Live Server, apri la cartella del progetto e avvia Live Server su `index.html`.

## Nota
Non serve installare nulla di speciale: il progetto è statico e si può usare direttamente in locale.
