# Installazione

Non c'è una vera "installazione": è un sito statico, quindi si tratta più che altro di scaricarlo e aprirlo.

## Cosa serve
Un browser recente (Chrome, Firefox, Edge o Safari vanno tutti bene) e una connessione a internet. La connessione serve perché Bootstrap arriva da un CDN e i dati meteo vengono chiesti in diretta a Open-Meteo.

## Per farlo partire
Se hai git, clona la repository:

```bash
git clone https://github.com/Halocavo92/DashBoard-Meteo.git
cd DashBoard-Meteo
```

Poi apri `index.html` nel browser e ci sei. In alternativa puoi anche scaricare lo zip da GitHub e aprire il file allo stesso modo, senza usare git.

## Se usi VS Code
Io mi trovo meglio con l'estensione Live Server: apri la cartella del progetto, clic destro su `index.html` e "Open with Live Server". Così a ogni salvataggio la pagina si aggiorna da sola, senza ricaricare a mano.

## Una precisazione
Trattandosi di un progetto statico, non c'è niente da compilare o da installare con npm. Si apre e funziona subito.
